Hive -- a powerful geometry library

1. What's Hive ?
    Hive is a powerful geometry library for C++. It can help you work out geometry problems.

2. What does Hive include?
    Up to now , Hive includes several headers:
        Zmath.h -- for basic math functions;
        Point.h -- defines the Point class;
        Line.h -- defines the Line class;
    
    We are going to make some other headers , including Circle,Triangle,Paradose ,etc.
